# React Markdown Editor

---

### Simple Editor, Simple Description

A simple markdown renderer web application made using TypeScript and React + Vite.

![desktop-preview](/image.png)



---

### How to run:

1. clone repo
2. `npm install`
3. `npm run dev`

---



### Further Extensions:

- [ ] Firebase integration (allowing saving of documents to the cloud)
- [ ] User sign in, to allow individual users to have a different store of documents
- [ ] Real time collaboration between users
